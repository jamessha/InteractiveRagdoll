An interactive ragdoll for you to love and/or abuse

Controls:
WASD - move
Mouse - look
Left Mouse/Space - primary
Right Mouse/B - secondary
1 - laser
2 - rockets
3 - gravity gun
R - reset
N - add new buddy
F - toggle wireframe mode
C - toggle angle constraints
V - toggle rotation contraints
